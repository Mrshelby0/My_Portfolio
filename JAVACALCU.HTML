Introduction

In this blog post, we'll create a simple calculator in Java that can perform basic operations such as addition, subtraction, multiplication, and division. Each operation will be encapsulated in its own method, making the code modular and easy to understand.

Code Overview

This Java program defines a simple calculator that can perform four basic operations: addition, subtraction, multiplication, and division. Each operation is implemented as a separate method. The user is prompted to choose an operation, and the corresponding method is called to perform the calculation.

Detailed Explanation

1. Import Statements

import java.util.*;

The import java.util.* statement imports the Java Utility package, which includes the Scanner class used for taking user input.

2. Class Definition

public class calculatorwithfunction {

The class calculatorwithfunction is defined as public, meaning it can be accessed from outside its package.

3. Sum Method

public static int sum(){
    Scanner s = new Scanner(System.in);
    System.out.println("Enter the value of a : ");
    int a = s.nextInt();
    System.out.println("Enter the value of B : ");
    int b = s.nextInt();
    int c = a + b;
    return c;
}

This method prompts the user to enter two integers a and b. It calculates their sum and returns the result.

4. Subtraction Method

public static int sub(){
    Scanner s = new Scanner(System.in);
    System.out.println("Enter the value of a : ");
    int a = s.nextInt();
    System.out.println("Enter the value of B : ");
    int b = s.nextInt();
    int c = a - b;
    return c;
}

Similar to the sum method, this method prompts the user for two integers and returns their difference.

5. Multiplication Method

public static int MULTIPLICATION(){
    Scanner s = new Scanner(System.in);
    System.out.println("Enter the value of a : ");
    int a = s.nextInt();
    System.out.println("Enter the value of B : ");
    int b = s.nextInt();
    int c = a * b;
    return c;
}

This method prompts the user for two integers and returns their product.

6. Division Method

public static int DIVISION(){
    Scanner s = new Scanner(System.in);
    System.out.println("Enter the value of divident : ");
    int a = s.nextInt();
    System.out.println("Enter the value of devisor : ");
    int b = s.nextInt();
    int c = a / b;
    return c;
}

This method prompts the user for two integers and returns their quotient.

7. Main Method

public static void main(String[] args) {
    System.out.println("Which operation do you want to do : ");
    System.out.println("1) SUM");
    System.out.println("2) SUBTRACTION");
    System.out.println("3) MULTIPLICATION");
    System.out.println("4) DIVISION");
    System.out.println("5) END");
    System.out.println(" >>>>>");

    Scanner s = new Scanner(System.in);
    int ans = s.nextInt();
    
    if (ans == 1) {
        System.out.println("The sum of A + B is : " + sum());
    }

    if (ans == 2) {
        System.out.println("The subtraction of A - B is :" + sub());
    }
    
    if (ans == 3) {
        System.out.println("The Multiplication of A AND B IS :" + MULTIPLICATION());
    }

    if (ans == 4) {
        System.out.println("The Division of A and B is : " + DIVISION());            
    }

    if (ans == 5) {
        System.out.println("Thank you for choosing this program...");
    }
}

The main method starts by displaying a menu of operations that the user can choose from. It then reads the user's choice using the Scanner class. Based on the user's choice, it calls the corresponding method (sum, sub, MULTIPLICATION, or DIVISION) and displays the result. If the user chooses option 5, the program prints a thank-you message and ends.

How the Program Works

  1. The program displays a menu of operations.
  2. The user selects an operation by entering a number (1 to 5).
  3. The program prompts the user to enter two integers.
  4. The selected operation is performed, and the result is displayed.
  5. If the user selects 5, the program prints a thank-you message and terminates.

Conclusion

This simple calculator program demonstrates basic user input handling and function usage in Java. Each operation is encapsulated in its own method, making the code modular and easy to understand. This is a great starting point for anyone looking to learn Java programming and understand how to work with user inputs and functions.
